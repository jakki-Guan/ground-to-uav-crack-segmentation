# Figure Generation Entry Points

This directory exposes the public figure-generation entrypoints used by the paper-facing assets.

- `paper_figures.py`: reusable helper module for dataset-overview, B1-mechanism, and B2-recovery figures.
- `generate_kaggle_progression_figure.py`: fixed Kaggle qualitative progression export.
- `export_jstars_safe_figures.py`: high-resolution figure bundle export.
- `make_seed_stability_report.py`: aggregate seed-stability report assets.
- `make_retention_report_assets.py`: source-retention and target bootstrap asset generation.
